# Seijin Photo 2026

## 概要
Seijin Photo 2026 は、二十歳の集い（同級会）向けに制作したリアルタイム写真投稿アプリです。  
参加者がスマートフォンから投稿した写真を、会場スクリーンに自動表示できるように設計しました。

本アプリは、実際のイベント利用を想定して設計・開発・運用まで一人で行いました。

---

## 制作背景
成人式後の同級会にて、参加者同士がその場で撮影した写真をリアルタイムで共有し、
会場のスクリーンに表示できる仕組みがあれば盛り上がると考え、本アプリを制作しました。

実際に約30名規模のイベントで運用し、約70枚の写真が投稿されました。

---

## 主な機能

### 参加者向け機能
- QRコード経由でログイン
- 写真投稿（1枚ずつ）
- コメント入力（任意）
- 投稿一覧表示
- 写真保存
- いいね機能（1セッション1回、トグル式）

### スクリーン表示機能
- 投稿写真の自動表示
- 下から上へ流れるスライド表示
- 新着写真のピックアップ表示
- 新規投稿がない場合のランダム表示
- ポーリングによる自動更新（1秒間隔）

### 管理者向け機能
- 管理者ログイン
- 投稿一覧確認
- 投稿削除
- 画像の単体ダウンロード
- コメント確認

## 実行環境について
本プロジェクトは、Python の仮想環境上での実行を想定しています。  
ローカル環境で動作確認を行う場合は、仮想環境を作成・有効化した上で実行してください。

---

## 実行方法

### 1. 仮想環境の作成（例）
```bash
python -m venv venv
```

### 2. 仮想環境の有効化
```bash
# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. ライブラリのインストール
```bash
pip install -r requirements.txt
```

### 4. 開発サーバー起動
```bash
python manage.py runserver
```
